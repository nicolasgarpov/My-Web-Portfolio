**README - My Cronometer**

### Description:
This is a simple web-based stopwatch application built using HTML, CSS, and JavaScript. It provides a user-friendly interface to track elapsed time, featuring start/pause and reset functionalities.

### Features:
- **Live Stopwatch Display:** The main screen prominently displays the elapsed time in the format HH:MM:SS.
- **Start/Pause Button:** Use the "Play" button (icon: `bi-play-fill`) to start or pause the stopwatch.
- **Reset Button:** The "Reset" button (icon: `bi-arrow-counterclockwise`) allows you to reset the stopwatch to 00:00:00.

### Usage:
1. Open the HTML file (`index.html`) in a web browser.
2. Click the "Play" button to start the stopwatch.
3. Use the "Pause" button to pause the stopwatch if needed.
4. Click the "Reset" button to reset the stopwatch to 00:00:00.

### External Libraries:
- **Bootstrap Icons:** Icons are sourced from the Bootstrap Icons library (v1.11.1).

### Fonts:
- **Google Fonts:** The fonts used in this project are Courgette and Satisfy.

### Favicon:
- The favicon features a stopwatch icon for a cohesive visual identity.

**README - Mi Cronómetro**

### Descripción:
Este es un sencillo cronómetro basado en la web, construido con HTML, CSS y JavaScript. Proporciona una interfaz amigable para rastrear el tiempo transcurrido, con funciones de inicio/pausa y reinicio.

### Características:
- **Visualización en Tiempo Real:** La pantalla principal muestra de manera destacada el tiempo transcurrido en el formato HH:MM:SS.
- **Botón de Inicio/Pausa:** Utiliza el botón "Play" (ícono: `bi-play-fill`) para iniciar o pausar el cronómetro.
- **Botón de Reinicio:** El botón "Reset" (ícono: `bi-arrow-counterclockwise`) te permite reiniciar el cronómetro a 00:00:00.

### Uso:
1. Abre el archivo HTML (`index.html`) en un navegador web.
2. Haz clic en el botón "Play" para iniciar el cronómetro.
3. Usa el botón "Pause" para pausar el cronómetro si es necesario.
4. Haz clic en el botón "Reset" para reiniciar el cronómetro a 00:00:00.

### Bibliotecas Externas:
- **Bootstrap Icons:** Los íconos provienen de la biblioteca Bootstrap Icons (v1.11.1).

### Fuentes:
- **Google Fonts:** Las fuentes utilizadas en este proyecto son Courgette y Satisfy.

### Favicon:
- El favicon presenta un ícono de cronómetro para una identidad visual cohesiva.
