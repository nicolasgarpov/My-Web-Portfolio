**README - RGB Color Picker**

### Description:
This web application enables users to interactively select RGB colors using sliders. The project is implemented using HTML, CSS, and JavaScript.

### Features:
- **RGB Color Sliders:** Three sliders (Red, Green, Blue) allow users to dynamically adjust color components.
- **Real-time Color Display:** The selected color is displayed in real-time as sliders are adjusted.

### Usage:
1. Open the HTML file (`index.html`) in a web browser.
2. Adjust the sliders for Red (R), Green (G), and Blue (B) to create the desired color.
3. Observe the selected color and its RGB values displayed in real-time.

**README - Selector de Color RGB**

### Descripción:
Esta aplicación web permite a los usuarios seleccionar colores RGB de forma interactiva mediante deslizadores. El proyecto está implementado utilizando HTML, CSS y JavaScript.

### Características:
- **Deslizadores de Color RGB:** Tres deslizadores (Rojo, Verde, Azul) permiten a los usuarios ajustar dinámicamente los componentes de color.
- **Visualización en Tiempo Real:** El color seleccionado se muestra en tiempo real a medida que se ajustan los deslizadores.

### Uso:
1. Abre el archivo HTML (`index.html`) en un navegador web.
2. Ajusta los deslizadores para Rojo (R), Verde (G) y Azul (B) para crear el color deseado.
3. Observa el color seleccionado y sus valores RGB que se muestran en tiempo real.
