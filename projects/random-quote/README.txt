**README - Random Quotes Generator**

### Description:
This is a web application that displays random quotes about determination. The application is built using HTML, CSS, JavaScript, and utilizes the Bootstrap framework for styling.

### Features:
- **Random Quotes Display:** The main screen prominently displays a random quote related to determination.
- **Author Display:** The author of the displayed quote is also shown.
- **Change Quote Button:** Click the "Next Quote" button to display a new random quote.

### External Libraries:
- **Bootstrap:** Styling is implemented using Bootstrap v5.3.2 for a sleek and responsive design.
- **Google Fonts:** The font used in this project is Courgette, sourced from Google Fonts.

### Favicon:
- The favicon features an image of a quote for a distinctive visual identity.

### Usage:
1. Open the HTML file (`index.html`) in a web browser.
2. The current random quote and its author are displayed.
3. Click the "Next Quote" button to display a new random quote.

**README - Generador de Citas Aleatorias**

### Descripción:
Esta es una aplicación web que muestra citas aleatorias sobre la determinación. La aplicación está construida utilizando HTML, CSS, JavaScript y utiliza el framework Bootstrap para el estilo.

### Características:
- **Visualización de Citas Aleatorias:** La pantalla principal muestra de manera prominente una cita aleatoria relacionada con la determinación.
- **Visualización del Autor:** También se muestra el autor de la cita actual.
- **Botón para Cambiar Cita:** Haz clic en el botón "Siguiente Cita" para mostrar una nueva cita aleatoria.

### Bibliotecas Externas:
- **Bootstrap:** El estilo está implementado utilizando Bootstrap v5.3.2 para un diseño elegante y receptivo.
- **Google Fonts:** La fuente utilizada en este proyecto es Courgette, obtenida de Google Fonts.

### Favicon:
- El favicon presenta una imagen de una cita para una identidad visual distintiva.

### Uso:
1. Abre el archivo HTML (`index.html`) en un navegador web.
2. La cita aleatoria actual y su autor se muestran en la pantalla.
3. Haz clic en el botón "Siguiente Cita" para mostrar una nueva cita aleatoria.
