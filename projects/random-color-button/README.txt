**README - Random Color Generator**

### Description:
This is a simple web application that generates random colors with a click of a button. The application is built using HTML, CSS, and JavaScript.

### Features:
- **Random Color Display:** The main screen prominently displays a randomly generated color code in the format #RRGGBB.
- **Change Color Button:** Click the "Change Color" button to generate a new random color.

### Usage:
1. Open the HTML file (`index.html`) in a web browser.
2. The current random color is displayed with its corresponding color code.
3. Click the "Change Color" button to generate a new random color.

**README - Generador de Colores Aleatorios**

### Description:
Este es una aplicación web sencilla que genera colores aleatorios con un clic de botón. La aplicación está construida utilizando HTML, CSS y JavaScript.

### Características:
- **Visualización de Color Aleatorio:** La pantalla principal muestra de manera prominente un código de color generado aleatoriamente en el formato #RRGGBB.
- **Botón Cambiar Color:** Haz clic en el botón "Cambiar Color" para generar un nuevo color aleatorio.

### Uso:
1. Abre el archivo HTML (`index.html`) en un navegador web.
2. El color aleatorio actual se muestra con su código de color correspondiente.
3. Haz clic en el botón "Cambiar Color" para generar un nuevo color aleatorio.
