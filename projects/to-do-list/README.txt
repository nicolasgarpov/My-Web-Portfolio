**README - Task List**

### Description:
This is a web application for managing your task list. The application is built in Spanish using HTML, CSS, and JavaScript. Bootstrap Icons and Google Fonts are incorporated for an aesthetically pleasing visual experience.

### Features:
- **Task Creation:** Enter a task in the "Enter Task" field and use the "Create Task" button.
- **Task List:** All created tasks will be displayed in the "My Tasks" section.

### Technologies Used:
- **Bootstrap Icons:** Bootstrap Icons (v1.11.1) are used to enhance the interface.
- **Google Fonts:** Courgette, Fredoka, and Satisfy fonts from Google Fonts provide a unique aesthetic.
- **Favicon:** The application's icon features a representative image of a task list.

### Usage:
1. Open the HTML file (`index.html`) in a web browser.
2. Enter a task in the "Enter Task" field and click "Create Task."
3. View all your tasks in the "My Tasks" section.

**README - Lista de Tareas**

### Descripción:
Esta es una aplicación web para gestionar tu lista de tareas. La aplicación está construida en español utilizando HTML, CSS y JavaScript. Se utilizan iconos de Bootstrap Icons y fuentes de Google Fonts para una experiencia visual agradable.

### Características:
- **Creación de Tareas:** Ingresa una tarea en el campo "Ingresar Tarea" y utiliza el botón "Crear Tarea".
- **Lista de Tareas:** Todas las tareas creadas se mostrarán en la sección "Mis Tareas".

### Tecnologías Utilizadas:
- **Bootstrap Icons:** Se utilizan iconos de Bootstrap Icons (v1.11.1) para mejorar la interfaz.
- **Google Fonts:** Las fuentes Courgette, Fredoka, y Satisfy de Google Fonts proporcionan una estética única.
- **Favicon:** El ícono de la aplicación presenta una imagen representativa de una lista de tareas.

### Uso:
1. Abre el archivo HTML (`index.html`) en un navegador web.
2. Ingresa una tarea en el campo "Ingresar Tarea" y haz clic en "Crear Tarea".
3. Observa todas tus tareas en la sección "Mis Tareas".
